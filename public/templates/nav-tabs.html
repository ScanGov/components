Nav tabs & pills

Three distinct navigation patterns in use across ScanGov sites — each styled differently via scangov.css. Choose by context. Icon colors inside nav-tabs and nav-pills are set per icon via 100+ .fa-* rules in scangov.css (dark and light mode variants) — they are inherited automatically; don't override them inline.

Top-level site nav (nav-tabs topnav)

Used for the cross-site navigation bar at the top of scangov.org and related sites. Tabs have a bottom-open border, rounded top corners, and an active bright-primary fill. Source: _includes/nav.html.

  • Rankings
  • Standards
  • Guidance
  • Data 
<ul class="nav nav-tabs topnav small mb-0">
  <li class="nav-item">
    <a class="nav-link active" aria-current="page" href="...">
      <i class="fa-solid fa-bars-progress" aria-hidden="true"></i> Label
    </a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="...">Label</a>
  </li>
</ul>

Topic filter tabs (nav-tabs filter)

Used for indicator/topic switching within a page (rankings, profile details, standards). Rendered via the _includes/filter-pill.html partial — set pillTitle, pillUrl, pillIcon, and pillActive before including. Active tab gets a bright-primary fill.

  • All
  • Accessibility
  • Botability
  • Security
  • Usability 
<ul class="nav nav-tabs filter small mb-0">
  <li class="nav-item">
    <a class="nav-link active mx-1" aria-current="page" href="...">
      <i class="fa-solid fa-circle-notch" aria-hidden="true"></i> All
    </a>
  </li>
  <li class="nav-item">
    <a class="nav-link mx-1" href="...">
      <i class="fa-solid fa-{icon}" aria-hidden="true"></i> Label
    </a>
  </li>
</ul>

Segmented control (nav-pills filter)

Use for compact multi-segment switches — e.g., All / Cities / States / Federal in the rankings. Pill links without bottom-open borders; active state fills primary color. No icon typically needed.

  • All
  • Cities
  • States
  • Federal 
<ul class="nav nav-pills filter small mb-0">
  <li class="nav-item">
    <a class="nav-link active mx-1" aria-current="page" href="...">All</a>
  </li>
  <li class="nav-item">
    <a class="nav-link mx-1" href="...">Cities</a>
  </li>
</ul>

Icon coloring

Icons inside nav-tabs and nav-pills receive automatic per-indicator colors (e.g., accessibility icon is one color, security another) defined by 100+ .fa-* rules in scangov.css. Both dark and light mode variants are included. Do not set icon colors inline — they are applied automatically once the correct Font Awesome class is used. See the /* Per-icon colors */ sections in scangov.css for the full list.
