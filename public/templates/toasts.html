Toasts & loading

Patterns for communicating in-progress and completed actions to users. All use proper ARIA attributes so screen readers announce them without focus movement.

Spinner (block)

Use .spinner-border with role="status" and a .visually-hidden label for screen readers. Pair with aria-busy="true" on the loading container when replacing content.

Loading…
Loading… 
<div class="spinner-border" role="status">
  <span class="visually-hidden">Loading…</span>
</div>

<!-- Small variant -->
<div class="spinner-border spinner-border-sm" role="status">
  <span class="visually-hidden">Loading…</span>
</div>

Icon spin (inline / button)

For buttons that trigger an async action, swap the button's icon to a spinning fa-spinner fa-spin while the action is pending. Restore the original icon on completion. Don't disable the button unless necessary — prefer visual feedback only. 

<!-- Pending state -->
<button class="btn btn-primary" disabled>
  <i class="fa-solid fa-spinner fa-spin me-2" aria-hidden="true"></i>Scanning…
</button>

<!-- Idle state -->
<button class="btn btn-primary">
  <i class="fa-solid fa-rotate-right me-2" aria-hidden="true"></i>Re-scan
</button>

Toast notifications

Use Bootstrap's .toast component for non-blocking status messages. Always include role="alert" aria-live="assertive" aria-atomic="true" so screen readers announce the message immediately. Position with .position-fixed .bottom-0 .end-0 for the standard bottom-right placement. Auto-dismiss after 5 seconds via data-bs-delay="5000" data-bs-autohide="true".

Done
Scan complete for iowa.gov.
Error
Could not reach iowa.gov. Try again. 
<!-- Mount inside a .position-fixed.bottom-0.end-0.p-3 container -->
<div class="toast" role="alert" aria-live="assertive" aria-atomic="true"
     data-bs-autohide="true" data-bs-delay="5000">
  <div class="toast-header">
    <i class="fa-solid fa-circle-check text-success me-2" aria-hidden="true"></i>
    <strong class="me-auto">Done</strong>
    <button type="button" class="btn-close" aria-label="Close"></button>
  </div>
  <div class="toast-body small">Scan complete for domain.gov.</div>
</div>

<script>
// Initialize and show
const el = document.querySelector('.toast');
const toast = new bootstrap.Toast(el);
toast.show();
</script>
