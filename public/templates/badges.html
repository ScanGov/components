Badges

Grade and score badges use class="badge text-bg-{color}". scangov.css overrides Bootstrap's success/warning/danger colors with WCAG 2.2 AA-verified values — don't hard-code the hex values in markup, just use the class.

Grade badges

Scores 0–100 map onto letter grades (A–F) and badge colors via two separate scales. The color scale is coarser than the grade scale: only 3 colors regardless of the 5-letter range. Use this mapping consistently across all sites.

Score-to-grade-to-color mapping
Score Grade Badge color Example
90–100 A text-bg-success A
80–89 B text-bg-warning B
70–79 C text-bg-warning C
60–69 D text-bg-danger D
0–59 F text-bg-danger F

Status badges

Use the same color tokens for non-grade status indicators (pass/fail, active/inactive, etc.). The same WCAG-verified colors apply.

Pass Partial Fail Active Inactive

Markup

<span class="badge text-bg-success">A</span>
<span class="badge text-bg-warning">C</span>
<span class="badge text-bg-danger">F</span>
<span class="badge text-bg-primary">Active</span>

Status icons

For inline pass/fail indicators within a table cell or text, use Font Awesome circle icons with text color utilities instead of a badge — these are inline, not block-level elements. Used in the rankings Status column and compliance tables. fa-circle-check text-success, fa-circle-xmark text-danger, and fa-circle-exclamation text-warning (for inaccessible/unknown state).

Pass Fail Unknown 
<i class="fa-solid fa-circle-check text-success" aria-hidden="true"></i>
<i class="fa-solid fa-circle-xmark text-danger" aria-hidden="true"></i>
<i class="fa-solid fa-circle-exclamation text-warning" aria-hidden="true"></i>
