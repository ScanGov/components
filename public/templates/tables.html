ScanGov

Tables

Reference markup for tables across ScanGov sites. scangov.css applies border, hover, and shadow styling to every .table automatically — don't add table-bordered, table-hover, or shadow classes. Always wrap the table in .table-responsive and give it a <caption class="visually-hidden"> describing its contents.

Standard

Default look: just class="table".

Example standard table
Site Organization Status
iowa.gov State of Iowa Active
hiv.gov Dept. of Health and Human Services Active

Sortable

Same class="table", plus th.sortable headers with sort-icon markup. See scangov/_includes/sortcontent.html and scangov/_includes/profile_table.html for the working click-to-sort JS — this is the static markup only.

Example sortable table
Rank Site Grade
1 iowa.gov A
2 hiv.gov C

Dense data

Add table-sm to opt a table out of the default hover/shadow styling — for raw data grids (CSV previews, data dictionaries) where that visual weight isn't wanted. Still wrap in .table-responsive.

Example dense data table
Field Type Description
urlkey string Primary domain for the site.
overallScore number Aggregate score across all indicators.
