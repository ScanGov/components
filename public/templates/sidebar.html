Sidebar

Vertical sidebar navigation using Bootstrap's .list-group with .list-group-item-action links. Use in a col-md-2 column alongside a col-md-10 main content column, with position-sticky to keep it in view while the content scrolls.

Sidebar with content area

The standard two-column layout: narrow sticky sidebar left, main content right. The active item uses .active and aria-current="page". Each icon is followed by a <small> label.
